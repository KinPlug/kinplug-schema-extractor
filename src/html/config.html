Schema Extractor

by KinPlug
v1.0.1

📊 What This Plugin Does

Adds an "Export App Schema" button to the record list view that extracts comprehensive app documentation to CSV.

✨ Exported Data

🏷️
App Info App ID & Name
📝
Field Details Label, Code, Type, Required, Options, Defaults
🔒
Unique Constraint Prohibit duplicate values setting
📑
Subtable Fields Nested fields with parent reference
🔄
Process Management Statuses & Actions
👥
Assignees Action & Status assignees

🚀 How to Use

  1. Save this plugin configuration
  2. Go to the app's record list view
  3. Click the "📊 Export App Schema" button in the header
  4. CSV file downloads automatically
ℹ️ No configuration required. Just save and start using!
